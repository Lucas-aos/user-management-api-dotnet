# User Management API

Simple REST API built with ASP.NET Core and PostgreSQL for managing users.

## Technologies

- C#
- ASP.NET Core
- Entity Framework Core
- PostgreSQL
- Swagger

## Features

- Create user
- Get all users
- Get user by ID
- Update user
- Delete user

## Endpoints

GET /api/users  
GET /api/users/{id}  
POST /api/users  
PUT /api/users/{id}  
DELETE /api/users/{id}

## How to run

1. Clone the repository
2. Configure the database connection in appsettings.json
3. Run migrations
4. Run the API
